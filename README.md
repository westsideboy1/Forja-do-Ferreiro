# Forja-do-Ferreiro
Um projeto crud para fixar conceitos, 100% em python.
O que o sistema faz(Sujeito a mudanças): O programa simula o livro de registros de um ferreiro de RPG. Ele permite catalogar as armas e armaduras que foram forjadas, controlar seus atributos (como dano e material) e gerenciar o estado de conservação de cada item (se está nova, gasta ou quebrada).
